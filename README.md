# ARK Road Lines Packers and Movers

A modern React-based website for ARK Road Lines Packers and Movers, featuring a built-in calculator for cost estimation.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Contact Information**: Easy access to phone and WhatsApp contacts
- **Service Overview**: Clear listing of services and working hours
- **Built-in Calculator**: Interactive calculator for quick calculations

## Tech Stack

- React 19
- Vite 7
- CSS3

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Anwarhacker/ARK.git
cd ARK
```

2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

Create a production build:
```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```
